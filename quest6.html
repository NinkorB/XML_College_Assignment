Paragraph 1

The Un-ordered list does not follow any specific sequence. The items or data sequence are not set by the developer. It can be displayed randomly by the browser.
Here is an example of some unordered list data:

Un-ordered List

  • Mango
  • Apple
  • Milk
  • Honey

This is a basic un-ordered list.

Paragraph 2

The Ordered list is helpful when we want to give step-by-step instructions to the user where order should be in sequence. The developer predefines the sequence of the order.
If we simply convert it to an ordered list by replacing the tags, the order will be fixed:

  1. Mango
  2. Apple
  3. Milk
  4. Honey
